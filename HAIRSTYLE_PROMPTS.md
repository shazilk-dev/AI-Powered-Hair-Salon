# AI Image Generation Prompts for Hairstyles

> Professional prompts for Leonardo.ai, Midjourney, Stable Diffusion, or Google Imagen

---

## Leonardo.ai Settings (Recommended)

- **Model:** Leonardo Phoenix or Leonardo Diffusion XL
- **Dimensions:** 832x1216 (portrait) or 896x1152
- **Preset:** Cinematic or Photography
- **Transparency:** Enable "Transparent Background" toggle if available
- **Negative Prompt:** face, eyes, nose, mouth, body, neck, shoulders, skin, portrait, person, human features, background, scenery

## Quick Tips for Better Results

### Leonardo.ai Specific:
- Use **Leonardo Phoenix** model for best transparency
- Set **Alchemy** to V2 for higher quality
- Enable **Photo Real** if available
- Use **Guidance Scale: 7-10** for better control
- Generate **4 variations** and pick best

### Midjourney Specific (if using):
- Add `--v 6` for latest version
- Add `--ar 2:3` for portrait aspect ratio
- Add `--no face body person` to negative prompt
- Use `--style raw` for photorealistic results

### If background isn't transparent:
- Use **remove.bg** (free: 50 images/month)
- Use **Photopea** (free online tool)
- Use **GIMP** with color-to-alpha feature
- Use **Adobe Express** background remover (free)

## Batch Generation Strategy

**Recommended order:**
1. Start with **OVAL** (easiest, most versatile)
2. Generate all **5 short/crop styles** across categories (consistent)
3. Generate all **long/layered styles** (consistent)
4. Generate specialty styles (pompadour, undercut, etc.)
5. Review and regenerate any that don't match requirements

**Time estimate:**
- With good prompts: ~30-45 minutes for all 30
- Including cleanup: ~2-3 hours total
