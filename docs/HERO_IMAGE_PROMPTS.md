# Hero Image Generation Prompts

## Context

These prompts are optimized for AI image generation (Gemini, Midjourney, DALL-E, Stable Diffusion) to create professional hero images for StyleAI hair salon application.

**Requirements:**

- **Aspect Ratio:** 4:5 (portrait) or 3:4 for mobile optimization
- **Resolution:** Minimum 2000x2500px (high DPI)
- **Style:** Photorealistic, professional, modern, clean
- **Color Palette:** Deep purples (#9333ea), blacks, whites, with warm skin tones
- **Mood:** Confident, futuristic, premium, accessible

---

## Prompt 1: AI-Powered Hair Consultation (Primary Hero)

```
Create a hyper-realistic photograph of a modern hair salon consultation scene with AI technology integration.

COMPOSITION:
- Medium shot, slightly angled from above at 15 degrees
- Subject: A confident person (diverse ethnicity, 25-35 years old) sitting in a premium salon chair, looking at a holographic transparent display showing different hairstyle options
- Background: Softly blurred modern minimalist salon interior with warm ambient lighting

TECHNICAL DETAILS:
- Camera: Shot on Sony A7R IV, 35mm f/1.8 lens
- Lighting: Soft key light from camera left, warm fill from right, subtle rim light on hair
- Color grading: Deep purple accent lights (hex #9333ea) mixed with warm golden salon lighting
- Depth of field: f/2.8 for soft background blur while keeping subject sharp

STYLE & MOOD:
- Professional photography, editorial quality
- Clean, uncluttered composition
- Modern luxury aesthetic
- Technology seamlessly integrated (holographic UI elements with purple glow)
- Warm and inviting, not cold or sterile

SPECIFIC ELEMENTS:
- Floating holographic interface showing 3-4 hairstyle previews in transparent purple-tinted panels
- Subtle digital particles or scan lines around the person's face (non-intrusive)
- Professional salon equipment visible but blurred in background
- Premium leather salon chair in deep charcoal or black
- Modern geometric mirrors with LED backlighting

AVOID:
- Overly futuristic sci-fi elements
- Neon colors (except subtle purple accents)
- Cluttered or busy backgrounds
- Unflattering angles or harsh lighting
- Stock photo look
```

## Implementation Notes

1. **File Naming Convention:**
   - `hero-salon-primary.jpg` (Prompt 1)
   - `hero-salon-tech.jpg` (Prompt 2)
   - `hero-salon-transform.jpg` (Prompt 3)
   - `hero-salon-interior.jpg` (Prompt 4)

2. **Optimization:**
   - Export at 2x size for Retina displays
   - WebP format for web (fallback to JPEG)
   - Compress to ~200-300KB without visible quality loss
   - Use Next.js Image component for automatic optimization

3. **Placement:**
   - Save final images to `/public/` directory
   - Reference as `/hero-salon-primary.jpg` in page.tsx
   - Consider multiple images for variety/AB testing

4. **Fallback Strategy:**
   - Current gradient fallback is already implemented
   - Consider using CSS background-image with gradient overlay
   - Load images with priority flag for LCP optimization

## Color Extraction Guide

**Primary Purple:** #9333ea (OKLCH: oklch(0.57 0.25 294))
**Dark Background:** #000000 to #0a0a0a
**Accent Gold:** #d4af37 (for warm lighting)
**Text White:** #ffffff with 0.9 opacity

Use these in post-processing to maintain brand consistency.

## Usage Instructions

1. **Copy entire prompt** from desired section
2. **Paste into AI image generator:**
   - Midjourney: `/imagine` command
   - DALL-E 3: ChatGPT Plus image generation
   - Stable Diffusion: Use positive + negative prompts
3. **Generate 4-6 variations** per prompt
4. **Select best image** based on:
   - Facial diversity and approachability
   - Lighting quality (soft, not harsh)
   - Technology integration (subtle, not overwhelming)
   - Brand color alignment (purple accents)
5. **Post-process:**
   - Resize to 2000x2500px (4:5 ratio)
   - Apply slight purple color grade in shadows
   - Export as WebP + JPEG fallback
   - Compress to target file size
