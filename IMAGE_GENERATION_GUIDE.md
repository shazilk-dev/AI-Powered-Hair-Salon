## ğŸ¨ Leonardo.ai Workflow (Recommended)

### Settings:
- **Model:** Leonardo Phoenix
- **Dimensions:** 832x1216 (portrait)
- **Alchemy:** V2
- **Transparency:** ON (if available)

### Process:
1. Open **HAIRSTYLE_PROMPTS.md**
2. Copy prompt for a hairstyle (e.g., "oval-1 Long Layers")
3. Paste into Leonardo.ai
4. Add universal negative prompt (from HAIRSTYLE_PROMPTS.md)
5. Generate (create 4 variations, pick best)
6. Download PNG
7. Save to `public/hairstyles/{category}/{filename}.png`

### Example:
```
Category: oval
Filename: long-layers.png
Save to: public/hairstyles/oval/long-layers.png
```

## ğŸ”§ Troubleshooting

### No transparent background?
**Fix:** Use remove.bg or Photopea
1. Go to remove.bg
2. Upload image
3. Download PNG with transparency
4. Save to correct folder

### File too large (> 500KB)?
**Fix:** Will be optimized by `process-images` script

### Wrong dimensions?
**Fix:** Will be resized by `process-images` script to 800x1200px

### Can't run scripts?
**Fix:** Make sure you ran `pnpm install` first

## ğŸ’¡ Pro Tips

1. **Generate in batches:** Do all "crop" styles together, all "layers" styles together
2. **Use consistent settings:** Don't change Leonardo settings between images
3. **Save originals:** Keep unprocessed versions as backup
4. **Check transparency:** Open in Photoshop/GIMP to verify alpha channel
5. **Iterate quickly:** If first generation isn't perfect, try again with adjusted prompt

## ğŸ†˜ Need Help?

### Prompt not working?
- Try the "Alternative prompt" provided for each style
- Adjust negative prompt
- Change guidance scale (7-10)

### Background not transparent?
- Enable transparency in generation settings
- Use remove.bg post-processing
- Check if PNG has alpha channel

### Image doesn't match style description?
- Re-read the style description in SPEC.md
- Use more specific keywords in prompt
- Generate 4 variations and pick closest match
